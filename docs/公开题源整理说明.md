# 公开题源整理说明

## 目标

把公开 benchmark / dataset 先整理成统一的候选题层，方便后续做三件事：

1. 选择哪些来源最适合映射到 A/B/C 模块
2. 对公开题做改写、重参数化、语境重构
3. 保留可追溯来源，满足研究记录和季度轮换需要

## 与正式题库的关系

这些候选题不是正式私有题库，原因如下：

- 当前项目文档明确要求正式题库不得直接复用公开 benchmark 原题
- 持续监测题库需要更强的防污染和跨时间可比性控制
- 安全题还需要额外伦理审查和评分稳定性验证

## 当前整理原则

- 优先抓取官方公开实现或官方公开数据集
- 每条记录保留 `source_name`、`source_url`、`source_dataset`、`original_id`
- 每条记录明确 `module_candidates`
- 每条记录默认 `direct_reuse_allowed = false`
- 每条记录附带 `rewrite_guidance`
